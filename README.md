# Jeremy Hayes

I build software across Windows desktop apps, modern web apps, Discord tooling, and gameplay systems. My current work is centered on secure file workflows, portfolio case studies, Cloudflare-backed dashboards, and Roblox systems.

Last updated: April 12, 2026

## Current Focus

- **FileLocker** - A WinUI 3 Windows encryption app for encrypting, decrypting, validating, and managing local files. The latest release is `v1.0.5.2`, with NSIS installer distribution, GitHub Releases update checks, guided Beginner / Intermediate / Advanced modes, custom encrypt output destinations, queue metrics, and Help menu shortcuts for install/update troubleshooting.
- **A.S.P.E.C.T Discord bot** - A Python `discord.py` bot with slash commands, Gemini-powered chat, local SQLite/JSON persistence, economy and shop systems, welcome/goodbye automation, reaction roles, booster data, ship/image commands, and a small aiohttp API for integrations.
- **A.S.P.E.C.T dashboard** - A Cloudflare Workers dashboard for Discord OAuth login, manageable-server discovery, Cloudflare D1 settings storage, welcome-flow editing, audit logging, and bot sync endpoints.
- **Portfolio-2** - A Next.js, React, TypeScript, Tailwind CSS, and Framer Motion portfolio built around structured case studies, project pages, metadata, sitemap freshness, and a Turnstile-protected contact flow.
- **Gameplay systems** - Roblox Studio and Luau work focused on reusable systems, collaboration, and atmospheric game interactions for projects like Vacancy Filled.
- **Privacy experiments** - PrivacyLens, an early WinUI 3 prototype for a capture -> detection -> overlay screen-redaction pipeline.
