## What I Build

- **Desktop software** with C#, WinUI 3, .NET, Windows App SDK, installers, and updater flows
- **Web apps** with Next.js, React, TypeScript, Tailwind CSS, and Cloudflare
- **Discord systems** with Python, `discord.py`, OAuth, APIs, and persistent server settings
- **Gameplay systems** in Luau for Roblox projects that need reusable logic instead of one-off scripts
- **Privacy and security experiments** around safer file handling, redaction, and user-facing controls

## Featured Projects

### FileLocker
[FileLocker](https://github.com/jeremymhayes/FileLocker) is a Windows desktop app for encrypting, decrypting, validating, and managing local files.

What it focuses on:
- Guided experience modes for Beginner, Intermediate, and Advanced users
- Drag and drop queue workflows with clearer counts, statuses, and item details
- Safer file handling with validation, temporary output files, optional verification, backups, and cleanup controls
- GitHub Releases based update checks and installer release flows
- Documentation that stays aligned with the app and release process

Links:
- [Repository](https://github.com/jeremymhayes/FileLocker)
- [Latest Release](https://github.com/jeremymhayes/FileLocker/releases/latest)

### A.S.P.E.C.T Bot and Dashboard
A.S.P.E.C.T is my Discord bot and dashboard system.

What it includes:
- Slash commands, welcome systems, reaction roles, utility features, and server tooling
- Dashboard sign in with Discord OAuth
- D1 backed server settings and sync endpoints
- Local persistence, integrations, and API based features

Link:
- [A.S.P.E.C.T Dashboard](https://aspectbot.app)
