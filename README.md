### [FileLocker](https://github.com/AspectOV/FileLocker)
A Windows desktop app for secure file and folder encryption.

**Built with:** C#, .NET, WinUI  
**Highlights:**
- AES-256-GCM encryption for secure file protection
- Argon2id-based password handling
- Clean desktop UI with drag-and-drop support
- Built as a native Windows application instead of a simple script wrapper

**Highlights:**
