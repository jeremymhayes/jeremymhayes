## What I Make

I work across a few different lanes, but they connect more than they might look like at first.

- Windows desktop apps with C#, WinUI 3, Windows App SDK, installers, and update flows.
- Personal web projects with Next.js, React, TypeScript, Tailwind CSS, and Cloudflare.
- Discord bot and dashboard systems with Python, `discord.py`, Cloudflare Workers, D1, OAuth, and sync APIs.
- Roblox gameplay systems with Luau and Roblox Studio.
- Early privacy and security experiments around safer file handling, redaction, and user-facing controls.

## Featured Work

### FileLocker

[FileLocker](https://github.com/jeremymhayes/FileLocker) is a Windows desktop app for encrypting, decrypting, validating, and managing local files. It is built around a WinUI 3 interface, practical safety controls, guided experience modes, NSIS installer releases, and GitHub Releases update checks.

What I have been improving:

- Beginner, Intermediate, and Advanced modes so the app can be approachable without hiding important controls.
- Drag-and-drop queue workflows with clearer file counts, root selection counts, total size, statuses, and item details.
- Custom encrypt output destinations, including validation and persistence.
- Safer file handling with preflight checks, temporary output files, optional verification, backups, and cleanup controls.
- GitHub Releases updater behavior, installer naming consistency, and troubleshooting shortcuts from the Help menu.
- Documentation that stays aligned with the app, installer, updater, and public release process.

Useful links:

- [FileLocker repository](https://github.com/jeremymhayes/FileLocker)
- [Latest FileLocker release](https://github.com/jeremymhayes/FileLocker/releases/latest)

### A.S.P.E.C.T Discord Bot and Dashboard

A.S.P.E.C.T is my Discord bot and dashboard work. The bot is built in Python with `discord.py`, and the dashboard runs on Cloudflare Workers with Discord OAuth and D1-backed server settings.

What it includes:

- Slash commands, server features, welcome and goodbye automation, booster data, reaction roles, and utility commands.
- Gemini-powered chat with per-user preferences and local conversation persistence.
- Economy and shop systems backed by SQLite/JSON storage.
- A small aiohttp API for integrations.
- Dashboard sign-in through Discord OAuth.
- Manageable-server discovery based on Discord permissions.
- D1-backed welcome settings with bot sync endpoints and audit logging.

Useful link:

- [A.S.P.E.C.T dashboard](https://aspectbot.app)
